# 🎵 MusicEz — Full-Stack Music Tracker

MusicEz is an intuitive, all-in-one web application designed for music enthusiasts to seamlessly track their favorite songs, albums, and artists. Built as a full-stack application, it utilizes a lightweight frontend connected to a robust Node.js/Express backend and a MongoDB database.

---

## 🚀 Tech Stack

* **Frontend:** HTML5, CSS3 (with custom animations), and Vanilla JavaScript
* **Backend:** Node.js & Express.js
* **Database:** MongoDB (NoSQL)
* **Communication:** RESTful API endpoints exchanging JSON data

---

## ✨ Features

* **Complete CRUD Operations:** Add, view, edit, and delete artists, albums, and songs.
* **Optimized Data Exchange:** Fast and reliable communication between the UI and database using structured JSON.
* **Modern User Interface:** A sleek design featuring brand-defining aesthetics, smooth transitions, and responsive layouts.
* **Live Database Viewing:** Instantly view and log your entire music library in clean, uniform data tables.

---

## 🖥️ Application Instructions

### 🔍 Browsing Your Library
Use the **Navigation Bar** at the top of the screen to toggle between **Artists**, **Albums**, and **Songs**. Each page displays your saved entries in a uniform, easy-to-read table.

---

### ➕ Adding a New Entry
1. Navigate to your desired category (**Artists**, **Albums**, or **Songs**).
2. Go to the input section at the top of the page.
3. Fill in the relevant details and click **Add Entry**.

---

### ✏️ Editing and Deleting
* **To Edit:** Click the orange **Edit** button next to any record. The app will guide you through updating the details.
* **To Delete:** Click the red **Delete** button next to a record to permanently remove it from your database.
* **To Sync (Albums):** In the Albums section, use the **Sync** button (which functions as the update button) to keep your records aligned.

## ⚙️ How to Run Locally

To get this full-stack application up and running on your machine, follow these steps.

### Prerequisites

Before starting, you must have **Node.js** installed on your system. 
* If you do not have it, download and install the LTS version from the official website: **[Node.js Official Downloads](https://nodejs.org/)**.
* To verify it is installed correctly, open your terminal (CMD) and run:
  ```bash
  node -v
  npm -v

1. **Step 1** - Clone the repository

   git clone [https://github.com/Cal800/MusicEz---Full-Stack-Application-.git](https://github.com/Cal800/MusicEz---Full-Stack-Application-.git)
   cd MusicEz---Full-Stack-Application-

2. **Step 2** - Set up and run the backend

   Navigate to the backend directory:
    ```bash
   cd backend
   ```

   Install the dependencies:
    ```bash
   npm install
   ```

   Start the server:
   ```bash
   node server.js
    ```

4. **Step 3** - Run the front end

   Navigate to the front end directory and launch the html file (double click to open)
